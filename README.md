# .dotfiles


## Dependencies

- git
- stow
- zsh
- tmux
- neovim
- lazygit
- ripgrep
- starship

## Installation


1. Install dependencies
2. `cd` to the home directory
3. `git clone https://github.com/oskarilindroos/.dotfiles.git` Clone the repository
4. `cd .dotfiles` Change directory
5. `stow .` Make symbolic links
